linux主机信息收集

系统架构

  1. uname -a
    打印系统信息
  2. cat /etc/issue
    文件/etc/issue是一个文本文件，其中包含要在登陆提示之前打印的消息或系统标识。
  3. cat /etc/-release
    cat /etc/    -release，/etc/redhat-release 文件包含一个被破解析以获取信息的描述行。
  4. cat /proc/version
    /proc/version指定了Linux内核的版本，用于编译内核的gcc的版本壹基金内核编译的时间。他还包含器的用户名。
  5. cat /proc/sys/kernel/version
    /proc/sys/kerne/中的文件可以用来调整和监视Linux内核操作中的各种活动。

进程

ps -ef
ps aux
例出当前进程快照
ls -al /proc/
/proc是非常特殊的，他也是一个虚拟文件系统。它有时被称为过程信息伪文件系统。 它不包含“真是”文件，而是包含运行时系统信息（例如系统内存，安装的设备，硬件配置）。
ls -al /proc/99
查看关于PID 99的信息。
top
top命令显示您的Linux机器的处理器活动，并显示实时管理的任务。 它会显示正在使用的处理器和内存以及运行进程等其他信息。
id
找到用户的UID或GID等信息
w
显示登录到Linux服务器的人员
whoami
显示当前用户名
last
显示最后一次登录的用户名
cat /etc/master.passwd
/etc /eertcmaster.passwd on BSD ststems
cat /etc/suders
cat /etc/suders 文件内容是使用sudo命令必须遵守的原则
sudo -V
打印sudo版本字符串

SSH服务信息

cat /.ssh/authorized_keys
使用公钥认证，认证实体具有公钥和私钥。
私钥保存在你登陆的计算机上，而公钥存储在要登陆的所有计算机上的 .ssh/authorized_keys文件中。
cat ~/.ssh/id_rsa.pub
RSA 私钥会保存在你的home目录中：ssh/id_rsa。
cat/.ssh/known_hosts
ssh会把每个你访问过计算机的公钥(public key)都记录在这个文件中
cat/.ssh/identity.pub
文件identity.pub包含你的公钥，可以将其添加到其他系统的authorized_keys文件中。
cat/.ssh/identity
ssh客户端允许您选择读取RSA或DSA身份验证标识（私钥）的文件。
cat/etc/ssh/ssh_config
OpenSSH SSH服务端配置文件
service –status-all
检查所有服务状态
systemctl -a
列出安装在文件系统中的所有单元
service servicemname start
systemctl start servicename
启动某个服务

  1. 安全
    iptables -L
    列出所有规则链。
    iptables -F
    删除选定规则链中得所有规则
    iptables -A INPUT -p icmp -m icmp -icmp-type 0 -j ACCEPT
    请执行iptables -p icmp-help 获得跟多信息。
    iptables -A INPUT -p tcp -m tcp -sport 80 -m state -state RELATED,ESTABLISHED -j ACCEPT
    允许从/到 dst 端口80得tcp连接
    iptables -A OUTPUT -p tcp -m tcp -dport 80 -m state -state NEW,RELATED,ESTABLISHED,ESTABLISHED -j ACCEPT
    允许从/到 dst 端口80得TCP连接
    iptables -A INPUT -p udp -m udp -sport 53 -m state -state RELATED,ESTABLISHED -j ACCEPT
    允许来自src端口80得udp连接
  2. 网络
    ipconfig -a
    显示当前可用得所有接口
    hostname
    显示或设置系统得主机名称
    dnsdomainname
    显示系统得DNS域名
    netstat -antp
    显示网络状态
    cat/etc/network/interfaces
    /etc/network/interfaces 文件包含网络接口配置信息
    cat/etc/sysconfig/network
    /etc/sysconfig/network 文件用于指定有关服务器上所需网络配置得信息
    cat/etc/networks
    /etc/networks 是一个简单得ASVII文件，描述这些网络得已知DARPA网络和符号名称
    find / -writable -type d 2>/dev/null
    找出可写出得文件夹
    find / -perm -222 -type d 2>/dev/null
    找出可写出得文件夹
    find / -perm -o w -type d 2>/dev/null
    找出可写出得文件夹
    find / -perm -o x -type d 2>/dev/null
    找出可写出得文件夹
    find / (-perm -o w -perm -o x) -type d 2>/dev/null
    找出可写可执行的文件夹
    find / -xdev -type d (-perm -0002 -a ! -perm -1000) -print
    找出可写得文件
    find /der -xdev (-nouser -o -hogroup)-print
    找出不是所有者得文件
